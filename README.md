# Banking-calculator-function

A beginner-friendly command-line calculator built with Python that performs basic arithmetic operations through reusable functions. Users can repeatedly perform calculations until they choose to exit the program.

This project was built to strengthen my understanding of Python functions, loops, conditional statements, user input, and basic program structure while creating a practical calculator application.

# Project Overview

The Accounting Calculator is an interactive command-line application that allows users to perform four fundamental arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division

The calculator uses separate functions for each mathematical operation, making the code more organized, reusable, and easier to maintain. After each calculation, users can choose whether to perform another calculation or exit the program.

Although simple, this project demonstrates important programming concepts that serve as building blocks for larger Python applications.

# Features

- Function-based program design
- Addition, subtraction, multiplication, and division
- Division-by-zero error handling
- User-friendly command-line interface
- Continuous calculations without restarting the program
- Simple menu-driven navigation

# How to Run the Project

Clone the repository

git clone https://github.com/yourusername/Accounting-Calculator.git

Navigate into the project folder

cd Accounting-Calculator

Run the Python script

python calculator.py

Alternatively, open the Jupyter Notebook file and run all cells.

# How to Use

1. Launch the program.
2. Select an operation from the menu:
   - Addition
   - Subtraction
   - Multiplication
   - Division
3. Enter the first number.
4. Enter the second number.
5. View the calculated result.
6. Choose whether to perform another calculation or exit the program.
